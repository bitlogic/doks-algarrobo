    Conocé más sobre gestión de ingresos

    Con esta funcionalidad de promociones comerciales podrás asociar descuentos y beneficios a los tickets de estudiantes según los criterios de aplicación que definas. Además, puedes consultar los tickets pendientes de pago y las promociones financieras y comerciales vinculadas con ellos.

    Promociones financieras#

    Consultar promociones financieras#

    Con esta funcionalidad podrás consultar las promociones financieras disponibles. Consultar →

    Editar promociones financieras#

    Con esta funcionalidad podrás editar las promociones financieras disponibles. Editar →

    Medios de pago#

    Consultar medios de pago#

    Editar medios de pago#

    Reportes#

    Consultar tickets#

    Con esta herramienta puedes consultar las promociones comerciales que se aplicarán a los tickets. Buscar →

    Obtener reporte de promociones comerciales#

    Con esta herramienta puedes obtener un reporte personalizado de las promociones comerciales que se aplicarán a los tickets. Buscar promociones →

    Promociones comerciales#

    Crear promociones comerciales#

    Con esta herramienta puedes configurar promociones comerciales que se aplicarán a los tickets. Crear →

    Buscar promociones comerciales#

    Con esta funcionalidad podrás buscar las promociones comerciales que se hayan generado anteriormente. Buscar →

    Editar promociones comerciales#

    Con esta funcionalidad puedes realizar los cambios que consideres necesarios tanto en la información general de una promoción, como en sus reglas o criterios relacionados. Editar →

    Becas y beneficios#

    Crear becas y beneficios#

    Con esta funcionalidad podrás registrar nuevas becas y beneficios. Crear →

    Buscar becas y beneficios#

    Con esta funcionalidad podrás consultar todas las becas y los beneficios que se han registrado en el sistema. Buscar →

    Editar becas y beneficios#

    Con esta funcionalidad podrás realizar cambios en becas y beneficios que se hayan creado en otro momentos. Editar →

    Asignar o rechazar el otorgamiento de becas y beneficios a estudiantes#

    Con esta funcionalidad podrás realizar de manera individual la asignación o rechazo de becas o beneficios a los estudiantes. Asignar o rechazar →

    Asignar o rechazar masivamente el otorgamiento de becas y beneficios a estudiantes#
